# Public source data

This directory contains the bibliography and the Table 2/3 records required to regenerate the public portal catalogs. Internal full-text caches, screening notes, and section-level audit evidence are intentionally not included in the public repository.

- `survey.bib` — bibliography entries referenced by the portal.
- `world_model_methods.csv` — Table 2 Research Works and functional roles.
- `world_model_benchmarks.csv` — Table 3 Evaluation Resources and structured details.
- `tab_world_model_benchmarks.tex` — Table 3 Evaluation Focus labels used for synchronization checks.
